# angular-redux-boilerplate

> A Angular(v1.5+)/Redux application boilerplate using material design

## Contains

- [x] [Angular](https://github.com/angular/angular.js)
- [x] [Angular Material](https://github.com/angular/material)
- [x] [Angular UI Router](https://github.com/angular-ui/ui-router)
- [x] [Redux](https://github.com/reactjs/redux)
- [x] [NG Redux](https://github.com/angular-redux/ng-redux)
- [x] [Redux UI Router](https://github.com/neilff/redux-ui-router)
- [x] [Redux Saga](https://github.com/yelouafi/redux-saga)
- [x] [Redux DevTools Extension](https://github.com/zalmoxisus/redux-devtools-extension)

## Usage

### Setup

```
$ npm install
```

### Developing

```
$ npm run dev
```

### Building

```
$ npm run build
```

### Running

```
$ npm start
```

### Testing

```
$ npm test
```
